# How to edit listing

1. Open the [ConfigCat monday.com developer](https://configcat.monday.com/apps/manage) page
2. Select the "ConfigCat Feature Flags" app. (Status should be "Marketplace")
3. On the left menu select the "Manage" > "Listing Page".
4. Click on the "New request" button on the right corner.

> **Note**
> When editing listings check the app "usage" page as well and update if necessary.

> **Note**
> The current images can be found on the GDrive > Design-Illustration-Graphics > integrations folder.

# Current listing values

**App Name:** ConfigCat Feature Flags

**App Values:** Popular with Devs

**Short Description:**
Manage ConfigCat feature flags from your monday dashboard.

**Description:**
Manage features and change your software configuration using **ConfigCat Feature Flags**, without the need to re-deploy code.

A 10 minute trainable dashboard allows even non-technical team members to manage application features. Supports A/B testing, soft launching or targeting a specific group of users first with new ideas. Deploy any time, release when confident. Open-source SDKs enable easy integration with any web, mobile or backend application.

ConfigCat Feature Flags monday app key features:
- Link ConfigCat feature flags to your monday items.
- Turn features On / Off right from a linked item on your monday board.
- Add targeting or percentage rules for your feature flags
- View linked monday items in ConfigCat dashboard and jump to the item directly.
